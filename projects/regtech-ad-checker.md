# 藥妝品廣告詞違規檢測（黑客松決賽＋獲獎）
> **Problem → Method → Experiments → Results → Demo/Repo**

**一句話摘要**：用可解釋的語意理解方式做違規文案分類，並提供 Web UI + Open API；加上爬蟲抓電商文案做回歸測試，最後在競賽獲獎。

---

## Problem（問題）
藥妝品廣告文案常踩到法規紅線（誇大療效、涉及醫療宣稱等），人工檢查費時且標準難一致。目標是做一個「可快速檢測、能說明原因、法規變更時好維護」的工具。

---

## Method（方法）
### 1) 可解釋的語意理解（不是純 embedding 相似度）
- 利用主辦單位提供的語意理解引擎，把文案做句型/詞性結構化後比對，可解釋、也利於法規變更時快速增修。

### 2) 產品化：Web UI + Open API
- 建立開放式 API，並用網頁把結果圖形化呈現（輸入文案 → 輸出違規類型/提示）
- 加入簡易爬蟲：從電商平台隨機抓文案，做回歸測試與持續改善

## Experiments（實驗）
- 以爬蟲抓取的隨機廣告文案進行測試（50 筆）

---

## Results（結果）
- 50 筆測試資料中，僅 2 筆在「涉及誇大或造假」類別出現偽陰性，其餘皆通過測試，成功率達九成。
- 競賽成果：獲得「法遵/公司治理特別獎」與「卓騰技術應用獎」。
